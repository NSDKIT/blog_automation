# メガネ記事案ジェネレーター - 要件定義書

## 1. アプリの目的
メガネ関連の記事を自動生成し、Shopifyに投稿するためのWebアプリケーション

## 2. 主要機能

### 2.1 記事生成機能
- キーワード、ターゲット層、記事の種類を入力して記事を生成
- Google検索結果を分析してSEO最適化された記事を作成
- 知識ベースから眼鏡生産者の想いを取得
- 画像を自動選定して記事に挿入
- Shopify形式に自動変換して投稿

### 2.2 ユーザー管理機能
- ユーザー認証（ログイン/ログアウト）
- ユーザーごとの記事履歴管理
- 権限管理（管理者/一般ユーザー）

### 2.3 記事管理機能
- 生成された記事の一覧表示
- 記事の編集・削除
- 記事のプレビュー
- 記事の再投稿

### 2.4 設定管理機能
- Google Sheets IDの設定
- Shopify API設定
- LLM API設定（OpenAI, Anthropic, Gemini等）
- 知識ベース設定

## 3. 非機能要件

### 3.1 パフォーマンス
- 記事生成時間：3分以内
- 同時処理：10ユーザーまで

### 3.2 セキュリティ
- APIキーの安全な管理
- ユーザー認証の実装
- HTTPS通信

### 3.3 可用性
- エラーハンドリング
- リトライ機能
- ログ記録

## 4. 技術要件
- フロントエンド：React + TypeScript
- バックエンド：Python (FastAPI)
- データベース：PostgreSQL
- 認証：JWT
- デプロイ：Docker + Docker Compose
