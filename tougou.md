### **判定基準の詳細**

```
判定ロジック:
┌─────────────┬──────────┬────────────┬──────────┐
│ 判定        │ 競合度   │ 難易度     │ 優先度   │
├─────────────┼──────────┼────────────┼──────────┤
│ 🟢 即攻略   │ LOW      │ 0-30       │ 高       │
│ 🟡 中期目標 │ MED      │ 31-50      │ 中       │
│ 🔴 長期目標 │ HIGH     │ 51-100     │ 低       │
└─────────────┴──────────┴────────────┴──────────┘

優先度スコア計算式:
score = (検索ボリューム × 商業価値係数) ÷ (難易度 + 10)

商業価値係数:
- CPC < $0.30: 1.0
- CPC $0.30-1.00: 1.2
- CPC > $1.00: 1.5

推奨順位:
現在のドメイン権威を考慮した到達可能順位の予測
（難易度が低いほど上位表示の可能性が高い）

```

### **フィルタリング・ソート機能**

```
フィルター項目:
├ 判定: 🟢即攻略 / 🟡中期 / 🔴長期 / すべて
├ 検索ボリューム: 10万以上 / 50万以上 / 100万以上
├ CPC: $0.5以下 / $0.5-1.0 / $1.0以上
├ 競合度: LOW / MED / HIGH
└ 難易度: 0-30 / 31-50 / 51-100

ソート項目:
├ 優先度スコア順（デフォルト）
├ 検索ボリューム順
├ 難易度順（昇順/降順）
├ CPC順
└ 推奨順位順

```
